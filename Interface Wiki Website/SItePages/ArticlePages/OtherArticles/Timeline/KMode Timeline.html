This page will be going over all the events in The Interface universe in order. I will do this as extensively as possible so some speculation will be included but this will be shown as speculation to the best of my ability.

Below is an explanation for how the timeline works.


All events will be in chronological order if possible.


If multiple events happen in the same year that will be numbered in the order they occur, explanations below for clarity.

Events that happen within an episode not a flashback will have the episode number after the year.

if the order is unclear or there are no other events then there will be no number to signify what the order they occurred in.

in cases where only the order of some events are clear, the events that have no clear order will share a number.


The date is as specific as it is written, explanations below for clarity.

If a year is stated with no additional information that is the exact year it happened.

If a year is said to be before a year it is that year or any point before.

If a year is said to be after a year it is that year or any point after.

If a year is said to be around a year it is near that year but may be a few years before or after.
